Halaman Home

Halaman Materi

Materi yang disajikan memiliki 3 bahasa yaitu HTML, Javascript, dan PHP.

Halaman Kuis

Terdapat halaman kuis untuk mengasah kemampuan setelah melakukan pembelajaran.

Halaman Code Edtior

Halaman Code Editor dapat diakses pada saat menekan tombol misalnya "Coba HTML" di halaman materi HTML untuk mencoba coding langsung.

Halaman Dashboard Admin

Ketika user admin login akan disajikan grafik jumlah kuis dan materi yang sudah dibuat.

Halaman Tambah Materi

Halaman admin untuk membuat materi baru.

Halaman Tambah Kuis

Halaman admin ketika PO yang sudah dibuat admin dan sudah melakukan deal pada user TNT.

Halaman Preview Mobil Masuk PDF

Halaman TNT Update PO
